# Database Manager | Next.js Version

Bu loyiha avvalgi HTML/JS versiyasining **Next.js 14** (App Router) asosidagi professional talqinidir.

## Xususiyatlari
- **React & Next.js**: Dinamik va tezkor navigatsiya.
- **Tailwind CSS**: Zamonaviy, premium va moslashuvchan dizayn.
- **Lucide Icons**: Chiroyli va engil vektorli ikonkalar.
- **Chart.js**: Professional ma'lumotlar tahlili va grafiklar.
- **Supabase**: Markazlashgan ma'lumotlar bazasi bilan integratsiya.

## Ishga tushirish (Setup)

1. **Node.js o'rnating:** Agar yo'q bo'lsa, [nodejs.org](https://nodejs.org/) saytidan yuklab oling.
2. **Terminalni oching:** `database_manager_next` papkasiga kiring.
3. **Kutubxonalarni o'rnating:**
   ```bash
   npm install
   ```
4. **Loyihani ishga tushiring:**
   ```bash
   npm run dev
   ```
5. **Brauzerda ko'ring:** [http://localhost:3000](http://localhost:3000)

## Arxitektura
- `src/app`: Sahifalar va routing (Dashboard, Aholi, Statistika).
- `src/components`: UI komponentlar (Sidebar, Charts, Cards).
- `src/lib`: Supabase klienti va konfiguratsiya.
- `src/globals.css`: Tailwind va umumiy stillar.
